# Prompts Used

The following key prompts were used during the development of this project:

## 1. Project Scaffolding
> "Initialize a Next.js 14 project with TypeScript, Tailwind CSS, and App Router in the current directory. Include Papa Parse, Lucide React, and Prisma for SQLite."

## 2. OpenRouter Integration
> "Create a server-side service to call OpenRouter API using Claude 3.5 Sonnet. The function should take CSV data as input and return markdown insights."

## 3. CSV Uploader Component
> "Build a React component for CSV file uploading with drag-and-drop support. Use Papa Parse to convert the file into a JSON object and pass it to a parent callback."

## 4. Prisma 7 Migration
> "How do I configure the datasource URL in Prisma 7 when it's no longer allowed in the schema.prisma file? Provide a solution using prisma.config.ts."

## 5. UI Layout
> "Design a modern, premium-looking landing page with a hero section and a two-column dashboard for data preview and insights display. Use a floating navigation bar at the bottom."
